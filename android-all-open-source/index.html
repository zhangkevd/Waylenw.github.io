概述

此博客初衷是记录一下自己开发过程后使用到或者收集到的一些轮子和开源项目，不追求全，只追求实用吧。会不断的更新,也是希望能继续完善吧。

数据库

  • ORMLite
  • GreenDaoMaster

框架对比
ORMLitehe和greenDaoMaster简单性能对比

图片加载

  • ImageLoader
    ImageLoader是目前Android平台上使用数量最多也是最为常见的图片加载框架。
    使用讲解:http://blog.csdn.net/github_25928675/article/details/46442037
  • picasso
    Picasso是美国一家支付公司Square推出的。它的特点是通过复杂的压缩来减少内存的消耗,自带内存和硬盘的二级缓存
    使用讲解:http://blog.csdn.net/github_25928675/article/details/46660179
  • Fresco
    Fresco是FaceBook推出的一款图片加载框架。自动释放内存。
    使用讲解:http://fresco-cn.org/docs/index.html

框架对比
Stackoverflow几个框架对比

工具类

  • LogUtil
    支持直接打印数据集合,如List、Set、Map、数组
  • DroidFix
    热修复的工具,修改补丁

插件

  • folding-plug
    布局文件分组

View

Loading

  • MetaballLoading
    水滴传递的效果
  • WhorlView
    漩涡风格的加载效果
  • AVLoadingIndicatorView
    Android Loading动画的一个集合
  • android-shapeLoadingView仿58loading效果

ViewPager

  • SpringIndicator
    viewapger指示器，像水滴流动的效果

    • ProductTour
      ViewPager的PageTransformer切换动画
    • VerticalVie向上滑动的viewpager,没有衔接效果()wPager

    指示器

    • FlycoTabLayout
      包含两种TabLayout.[SlidingTabLayout是基于PagerSlidingTabStrip的大量修改，新增了很多自定义的功能。CommonTabLayout:不同于SlidingTabLayout对ViewPager依赖,它是一个不依赖ViewPager可以与其他控件自由搭配使用的TabLayout.

    Dialog

    • FlycoDialog
      多种样式与动画效果的对话框

ListView

  • 瀑布流
  • IndexView
    索引控件可拖动，可根据列表滚动移动
  • SwipeMenuListView(右滑删除)

侧滑菜单

  • DragLayout 使用 support.v4 包下的 ViewDragHelper 实现 QQ5.0 侧滑
  • FlowingDrawer(具有弹性的侧滑菜单)

下拉上拉刷新

  • BGARefreshLayout-Android多种下拉刷新效果(支持上拉刷新)

    • UltimateRecyclerView
      支持ListView、RecycleView上拉刷新以及右滑动删除，支持拖动，动画，和和粘性头部。隐藏和显示Actionbar和ToolBar等
    • Ultra-Pull-To-Refresh
      此库是单纯的下拉刷新的组件,使用起来简单，API LEVEL >= 8可以ui样式十分的简单。还有md风格的下拉刷新样式哦。
